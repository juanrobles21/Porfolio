Details

En este proyecto, se presenta el desarrollo de un ChatBot para WhatsApp utilizando Node.js. El objetivo del ChatBot es proporcionar respuestas automáticas a los usuarios en función de ciertos comandos o palabras clave ingresadas por ellos. Para lograr esto, se aprovechará la API de WhatsApp Business API que permite la interacción programática con la plataforma de WhatsApp.

Características principales:
    1. Configuración del entorno: Se configurará un proyecto de Node.js y se instalarán las dependencias necesarias, como Express
    2. Registro de una cuenta de WhatsApp Business API: Para interactuar con la API de WhatsApp, se registrará una cuenta de WhatsApp Business API.
    3. Creación de rutas y controladores: Se definirán las rutas de la API web utilizando Express.js y se crearán los controladores para gestionar las solicitudes entrantes y salientes de WhatsApp.
    4.Procesamiento de mensajes: Cuando el ChatBot reciba un mensaje de WhatsApp, se analizará el contenido para identificar palabras clave o comandos específicos y se generará una respuesta acorde a la solicitud.
